Topic 7:  Coding-as-a-Service 
Major AI companies are rolling out Coding-as-a Service tools like Cursor’s Cloud Agent, OpenAI 
Codex Web, Github Copilot, Anthropic Claude on the Web, Google’s Gemini Code Assist. These 
Coding-as-a-Service tools use LLMs for code generation, bug detection or enable vibe coding. 
This project explores how to make Coding AI agents and creating a Coding-as-a-Service tool 
that enhances user experience, accuracy, productivity, safety, and performance. 
(i) 
Read UC Berkeley’s Ben Recht blog article on asking Claude Coder to build itself: 
https://www.argmin.net/p/secrets-of-intelligence-services. Study the open-sourced 
code at: https://github.com/benjamin-recht/microagent, and describe in your own 
words with visualization how an AI coding agent works in general with API calls.  

(ii) 
Extend the above Claude Coder or design your own into a Coding-as-a-Service.  

(iii) 
Describe how you evaluate your Coding-as-a-Service using performance metrics 
(e.g., Pass@k, CodeBLEU, functional correctness), see OpenAI paper “Evaluating 
Large Language Models Trained on Code” (https://arxiv.org/abs/2107.03374).  
 
(iv)
Show how your Coding-as-a-Service is useful to a SaaS user, e.g., allow someone to 
vibe-code, teach junior programmers to write code, do scientific computing etc.
